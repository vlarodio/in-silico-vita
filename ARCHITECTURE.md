# Архитектура In Silico Vita

Клеточная эволюционная симуляция. TypeScript, Canvas 2D, ноль зависимостей.

## Файловая структура

```
src/
├── types.ts         # Интерфейсы: CellStore, ResourceGrid, SimParams, SimState, хелперы
├── params.ts        # DEFAULT_PARAMS — все значения по умолчанию
├── index.ts         # Точка входа: createSimulation(container, options?)
├── engine.ts        # Главный цикл (requestAnimationFrame)
├── cellstore.ts     # Хранилище клеток (плоские типизированные массивы)
├── movement.ts      # Движение + хемотаксис (приоритеты целей)
├── metabolism.ts    # Энергия: метаболизм, питание из среды, смерть
├── resources.ts     # Ресурсная сетка («пастбища»): consumption, logistic regen
├── grid.ts          # Пространственная сетка для поиска соседей (O(1) на клетку)
├── behavior.ts      # Взаимодействия (налог + атаки) + деление с мутациями
├── events.ts        # Инфекции (скачки агрессии по расписанию)
├── render.ts        # Отрисовка: ресурсный фон (ImageData) + клетки (Canvas 2D)
├── stats.ts         # Сбор и отрисовка графиков (численность, средняя агрессия)
├── ui.ts            # DOM-панель управления (Start/Stop, Pause, Speed, поля ввода, Apply)
```

## Главный цикл (engine.ts → tick)

Каждый кадр, если `running && !paused`, выполняется `speed` тактов:

1. **moveCells** — случайное движение + хемотаксис
2. **metabolize** — energyCost, питание из ресурсной сетки, смерть при E ≤ 0
3. **regen** — логистическое восстановление ресурсов
4. **handleInteractions** — налог мирных защитникам + атаки (красный/синий бой по энергии)
5. **divideCells** — деление при E ≥ energyDiv, возраст ≥ minDivisionAge, мутация N(0,σ)
6. **applyInfections** — раз в infectionInterval тактов скачки агрессии
7. **collectStats** — запись среза в историю для графиков

После тактов: **onFrame** (обновление UI), **render** (основной canvas), **renderStats** (график).

## Данные

### CellStore — плоские типизированные массивы

```
x:        Float64Array[capacity]   — координата X
y:        Float64Array[capacity]   — координата Y
aggression: Uint8Array[capacity]   — ген агрессивности 0..100
energy:   Float64Array[capacity]   — запас энергии
age:      Uint32Array[capacity]    — возраст в тактах
alive:    Uint8Array[capacity]     — 1 жива, 0 мертва
count                              — текущее число живых (активный диапазон [0..count))
capacity                           — размер выделенных массивов (= maxCellCount)
```

Удаление клетки — swap с последней живой и count--. Активный диапазон всегда непрерывен.

### ResourceGrid — 2D-сетка ресурсов

```
data:  Float64Array[gridWidth * gridHeight]  — значения ресурса в ячейках
width, height                                 — размеры сетки (не пиксели)
```

Логистический реген: `resource += regenRate * (1 - resource / K)`.

### Пространственная сетка (grid.ts)

Мир разбит на ячейки размером `max(attackRadius, taxRadius)`. Ключ = `gy * gridW + gx`, значение = список индексов клеток в ячейке. Перестраивается каждый такт (O(n)). Поиск соседей: 3×3 ячейки + фильтр по тороидальному расстоянию.

## Поведение клеток

### Пороги агрессивности

| Диапазон | Тип | Цвет | Эффективность питания |
|---|---|---|---|
| 0–30 | Мирный | Зелёный | 1.0 |
| 31–70 | Защитник | Синий | 0.8 |
| 71–100 | Агрессор | Красный | 0.4 |

### Взаимодействия (при касании, радиус 4)

- **Мирный** → платит taxRate% энергии ближайшему защитнику
- **Защитник** → атакует ближайшего агрессора. Бой по энергии: умирает тот, у кого меньше
- **Агрессор** → атакует ближайшего. Против синего — бой по энергии, против других — мгновенное убийство

### Хемотаксис (приоритеты, радиус 60, bias 30%)

| Тип | Приоритет 1 | Приоритет 2 | Приоритет 3 |
|---|---|---|---|
| Зелёный | — | — | Еда |
| Синий | Агрессор | Мирный | Еда |
| Красный | Мирный | Еда | — |

### Деление

E ≥ 410 (0.8 × 512), age ≥ 20. Энергия пополам. Дочерняя агрессия = родитель + N(0, 5), обрезка [0, 100].

| minDivisionAge | 20 | Мин. возраст для деления |

| resourceRegenRate | 4 | Скорость регена |
| resourceK | 256 | Максимум ресурса в ячейке |
| baseSpeed | 1 | Скорость клеток |

| taxRadius / attackRadius | 4 | Радиус касания |
| taxRate | 0.1 | Доля энергии в налог |
| killAggressorReward | 50 | Награда агрессору за убийство |

| infectionInterval | 0 | Интервал инфекций (0 = выкл) |
